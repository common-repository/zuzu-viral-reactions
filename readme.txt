A simple and lightweight reactions plugin that allows people to express how they are feeling about your content.

== Description ==

= Main Plugin Features =

* Zuzu Viral Reactions adds responsive reaction buttons to your posts.
* Zuzu Viral Reactions Increase your blog engagement.
* Great user experience on both mobile & desktop.
* Zuzu Viral Reactions plugin remembers the reactions of every user to avoid spamming on your blog.


= A fun way to activate your readers =
People often don't have time to leave a comment and nothing's more frustrating than an empty comment section. Zuzu Viral Reactions activates even your busiest readers. All it takes is one click!


== Installation ==

1. Install the plugin through the WordPress plugins screen directly (Plugins - Add new - Search for "Zuzu Viral Reactions") or upload the plugin files to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' screen in WordPress.

When Zuzu Viral Reactions is activated, the reaction buttons are shown on each post by default.

In the settings page you have few options:

1. Show buttons by default or Don't show buttons by default. In each case you can override the global setting in the edit post view.
2. You also can change the reaction words according to your language.


**Short code**

You can always use short code `[zvr_reactions]` in the edit post screen. If you want to add Zuzu Viral Reactions to your template, use `if (function_exists('zvr_reactions')) { zvr_reactions() }`.


== Frequently Asked Questions ==

= Why should I use Zuzu Viral Reactions? =

People react up to 30 times more than they comment!

= Is it free? =

Yes, absolutely!

= Can I use different style smiley sets? =

Not yet :( But we're adding new styles options soon.

= Can I use short code? =
Yes!

= Does it work on multisites? =

Yes!

= Does it work with my theme? =

We've tested the plugin on various themes. Currently, Zuzu Viral Reactions looks great on white backgrounds.

= Which browsers are supported? =

All recent browsers are supported. Internet Explorer is supported from version IE9 onwards.

= The reaction buttons are visible on every post, can I choose on which posts they are shown? =

Yes, you can override the global setting in the edit post screen.

= My reactions don't show when I reload the page. What's up? =
Some WordPress installation cache the reactions for a while. Your reactions are saved and will show in a couple of minutes usually.
